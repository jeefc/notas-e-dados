# Notas e Dados 📝

Este é o repositório do meu blog pessoal, construído para documentar minha jornada técnica e reflexões sobre tecnologia.

## 🚀 Sobre o Projeto
O blog é um "caderno aberto" focado em:
- **Cloud & DevOps:** Estudos para certificações AWS, segurança e infraestrutura.
- **Desenvolvimento:** Projetos em Django, Next.js e experimentações com micro-SaaS.
- **Tecnologia & Sociedade:** Reflexões sobre o impacto social da tecnologia, filosofia e relações de trabalho.

## 🛠️ Stack Tecnológica
- **Gerador de Site Estático:** [Hugo](https://gohugo.io/)
- **Tema:** [PaperMod](https://github.com/adityatelange/hugo-PaperMod)
- **Hospedagem:** Cloudflare Pages
- **Workflow:** Deploy automático via GitHub

## 📂 Estrutura do Repositório
- `content/posts/`: Artigos e notas técnicas em Markdown.
- `content/sobre/`: Informações sobre o autor.
- `static/`: Imagens e outros arquivos estáticos.
- `hugo.yaml`: Arquivo principal de configuração do site.

## 💻 Desenvolvimento Local
Se você tiver o Hugo instalado (versão Extended recomendada), pode rodar o site localmente:

1. Clone o repositório:
   ```bash
   git clone https://github.com/jeefc/notas-e-dados.git
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   hugo server -D
   ```
3. Acesse `http://localhost:1313` no seu navegador.
